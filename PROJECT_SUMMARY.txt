HOW TO RUN THE PROJECT
====================================================================================================

QUICK START:
1. Open PowerShell in project directory
2. Run: .\install.ps1
3. Follow the prompts
4. Application starts automatically

MANUAL START:
1. Activate virtual environment: .\venv\Scripts\Activate.ps1
2. Ensure MongoDB is running: net start MongoDB
3. Run application: python run.py
4. Open browser: http://127.0.0.1:5000

FIRST USE:
1. Click "Register" and create an account
2. Login with your credentials
3. Add patient records using "Add Patient"
4. Manage patients (view, edit, delete)

RUN TESTS:
1. Activate virtual environment
2. Run: pytest -v
3. Or with coverage: pytest --cov=app tests/
